# Orion System 🚀

O **Orion System** é uma plataforma avançada de governança de TI que integra **ITSM (IT Service Management)** e **Monitoramento Ativo de Ativos** em uma única interface inteligente. 

Desenvolvido para oferecer máxima visibilidade operacional, o sistema combina a gestão de chamados com o acompanhamento em tempo real de hardware e performance através de um agente proprietário.

## ✨ Visão Geral & Diferenciais
- **Interface High-Tech**: Design moderno baseado em *glassmorphism* e micro-interações fluidas.
- **Ecossistema Unificado**: Integração nativa entre o portal de suporte e o monitoramento de máquinas.
- **Monitoramento Inteligente**: Coleta proativa de métricas essenciais (CPU, RAM, Disco) e execução de comandos remotos.
- **Segurança de Acesso**: Autenticação baseada em tokens de máquina para login simplificado e seguro.

## 🛠️ Stack Tecnológica
- **Frontend**: React + TypeScript + Vite.
- **Design System**: Tailwind CSS com componentes customizados de alta fidelidade.
- **Backend (API)**: Go (Chi framework) rodando em arquitetura serverless de alta performance.
- **Infraestrutura de Dados**: Supabase (PostgreSQL, Real-time, Auth).
- **Agente Nativo**: Orion Agent desenvolvido em Go para execução otimizada em ambientes Windows.
